## セットアップ (Setup)

### 開発環境の準備
1.  [Rust](https://www.rust-lang.org/ja/tools/install) をインストール。
2.  [Node.js](https://nodejs.org/) をインストール。
3.  プロジェクトをクローンまたはダウンロード。

### 実行・ビルド
```bash
# 依存関係のインストール
npm install

# 開発モードで起動
npm run tauri dev

# インストーラー (.msi) の作成
npm run tauri build
```
